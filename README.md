# SmileRT LIVE Manager

ライブイベントのセットリスト・タイムテーブル・出演者情報を一元管理するWebアプリケーションです。

## 機能

- **イベント管理** — イベントの作成・編集・削除
- **出演者管理** — 出演者の登録、ドラッグ＆ドロップによる順番入れ替え
- **タイムテーブル** — 開演時間に基づく自動タイムライン計算、行の入れ替え
- **出演者ポータル** — 出演者が自分でセットリストを入力できるページ
- **技術資料表示** — 音響・照明スタッフ向けの技術シート
- **CSV取込/書き出し** — Googleフォーム回答のCSV取込、Excel互換CSV書き出し
- **印刷対応** — 技術資料の印刷用レイアウト

## ページ構成

| ファイル | 説明 |
|---|---|
| `index.html` | トップページ（ダッシュボード） |
| `admin.html` | 管理画面（イベント・出演者・タイムテーブル管理） |
| `performer.html` | 出演者ポータル（イベント選択→出演者選択） |
| `input.html` | セットリスト入力フォーム |
| `view.html` | 技術資料表示・印刷・CSV書き出し |

## 使い方

1. `index.html` をブラウザで開く（ローカルサーバー推奨）
2. 管理画面でイベントを作成
3. 出演者を追加してセットリストを入力
4. 技術資料ページで確認・印刷・CSV書き出し

### ローカルサーバーでの起動

```bash
npx -y http-server . -p 8080 -c-1
```

ブラウザで `http://localhost:8080` にアクセス。

## 技術仕様

- **フロントエンドのみ** — HTML / CSS / JavaScript（フレームワーク不使用）
- **データ保存** — `localStorage`（サーバー不要）
- **レスポンシブ対応** — モバイル・デスクトップ両対応
